# Support

## Where to Get Help

Syncore uses different channels for different kinds of requests.

Use GitHub Issues for:

- reproducible bugs
- scoped feature requests
- regressions
- specific follow-up work tied to the repository

Use GitHub Discussions for:

- usage questions
- design feedback
- architecture tradeoffs
- early feature exploration
- general community conversation

Repository links:

- Issues: https://github.com/JPSAUD501/Syncore/issues
- Discussions: https://github.com/JPSAUD501/Syncore/discussions
- Security reporting guidance: [`SECURITY.md`](SECURITY.md)

## How to Ask a Good Question

When asking for help, include:

- what you are trying to do
- which package, example, or platform is involved
- exact command or code snippet
- current behavior
- expected behavior
- error output or screenshots if relevant

Good support requests are specific, reproducible, and scoped.
