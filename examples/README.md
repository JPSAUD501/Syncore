# Examples

These examples are reference integrations for the current Syncore targets:

- [`browser-esm`](browser-esm)
- [`electron`](electron)
- [`expo`](expo)
- [`next-pwa`](next-pwa)
- [`sveltekit`](sveltekit)

They are intentionally small and focused on runtime wiring, not polished UI.

Use the quickstarts first when you want a zero-to-running path. Use the examples when you want a slightly fuller integration that mirrors the smoke-tested workspace fixtures.
